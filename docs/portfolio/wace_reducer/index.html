Разработка волнового редуктора.

Разработка волнового двухступенчатоно редуктора повышенной надежности и износостойкости при нестационарной эксплуатации в условиях неблагоприятной среды. Приводом редуктора на входные валы являются 2 электродвигателя мощностью 2 кВт каждый. Общие характеристики редуктора:

  • Мощность = 4 кВт
  • Ресурс = 1000 часов
  • Вес = 26 кг
  • Частота вращения входных валов = 15 000 об/мин
  • Частота вращения выходного вала (обода) = 62,5 об/мин
  • Крутящий момент входного вала (одного) = 0,6365 N·m
  • Крутящий момент выходного вала (обода) = 587 N·m
  • Передаточное отношение = 240
